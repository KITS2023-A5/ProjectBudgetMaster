Naming convention: camelCase

**Quy tắc đặt tên branch:**

-   Tính năng mới: feature/{mô_tả_ngắn_gọn_feature}
-   Fix bug: bugfix/{mô_tả_ngắn_gọn_bug}
-   **_Lưu ý: Chỉ tạo branch mới từ main_**

**Trước khi commit:**
-   Format code 
-   Organize import
-   Xóa các lệnh System.out.println
